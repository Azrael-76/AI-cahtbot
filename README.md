# AI-chatbot

![alt text](image.png)

## 项目企划

- 基于百度智能云大模型ERNIE-4.0实现网页聊天机器人
- 支持键盘与语音交互
- 支持中文对话
- 支持上下文对话
- 支持多轮对话
- 支持自定义回复


## 项目技术栈

- Python
- Fastapi
- React + Vite
